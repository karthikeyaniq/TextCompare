TextCompare Privacy Policy

Overview

TextCompare is a text and code comparison tool that helps users identify differences between two pieces of text. This privacy policy explains how we handle your data.

Data Collection

We do not collect, store, or transmit any personal data to external servers.

TextCompare operates entirely within your browser and does not send any data to external servers or third parties.

Data Storage

The extension uses Chrome's local storage API to temporarily store:

  • Comparison data: Text content for comparison (temporarily stored)
  • User preferences: Settings like diff mode and view options
  • Clipboard content: Only when you explicitly choose to compare with clipboard

All stored data is:

  • Stored locally on your device
  • Automatically cleared after use
  • Never transmitted to external servers

Permissions Explained

  • activeTab: Access to the current tab to read selected text for comparison
  • clipboardRead: Read clipboard content when you choose "Compare with Clipboard"
  • contextMenus: Create right-click menu options for quick comparison
  • host_permissions: Access URLs to fetch content for comparison
  • scripting: Execute scripts to read selected text and fetch URL content
  • storage: Store temporary comparison data and user preferences
  • tabs: Access tab information to open comparator and read selected text

Data Usage

All data processing happens locally in your browser:

  • Text comparison algorithms run on your device
  • No data is sent to external servers
  • No analytics or tracking is performed
  • No user behavior is monitored

Third-Party Services

TextCompare does not use any third-party services, analytics, or external APIs. All functionality is self-contained within the extension.

Data Security

Your data is secure because:

  • All processing happens locally in your browser
  • No data leaves your device
  • Uses Chrome's secure storage APIs
  • No external network requests for data

User Control

You have complete control over your data:

  • All features require explicit user action
  • Stored data is automatically cleared after use
  • You can uninstall the extension to remove all stored data
  • No persistent data collection

Contact

If you have any questions about this privacy policy or how TextCompare handles your data, please contact us through the Chrome Web Store.

Updates

This privacy policy may be updated occasionally. Any changes will be reflected in the extension description on the Chrome Web Store.
